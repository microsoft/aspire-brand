# Aspire Brand Assets

<p align="center">
  Official brand assets, logos, colors, and presentation materials for <a href="https://aspire.dev">Aspire</a>.
</p>

This repository stays intentionally lightweight: curated SVG logo files, the current presentation deck, and an Astro-powered reference site that can publish to GitHub Pages after merge.

## What is Aspire?

[Aspire](https://aspire.dev) is a multi-language local dev-time orchestration tool chain for building, running, debugging, and deploying distributed applications.

## Repository contents

| Path | Description |
| --- | --- |
| [`logo/`](logo) | Official SVG logo files for reuse. |
| [`slides/`](slides) | Presentation materials for community talks, demos, and events. |
| [`src/`](src) | Astro source for the GitHub Pages brand site. |

## Working locally

```bash
npm install
npm run dev
npm run build
```

## Usage guidance

Use the provided SVG assets whenever possible for crisp scaling. Always refer to the product as **Aspire**, not ".NET Aspire". Keep the Aspire logo proportions and fill colors intact, avoid introducing alternate wordmarks or composite marks, and point people to [aspire.dev](https://aspire.dev) for official docs and product content.
