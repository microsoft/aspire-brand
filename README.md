# Aspire Brand Assets

<p align="center">
  Official brand assets, logos, colors, and presentation materials for <a href="https://aspire.dev">Aspire</a>.
</p>

This repository brings together the official Aspire logos, the core color references, and the current presentation deck in one place.

## What is Aspire?

[Aspire](https://aspire.dev) is a multi-language local dev-time orchestration tool chain for building, running, debugging, and deploying distributed applications.

## Repository contents

| Path | Description |
| --- | --- |
| [`logo/`](logo) | Official SVG logo files for reuse in docs, decks, sites, and events. |
| [`slides/`](slides) | Presentation materials for talks, demos, and community sessions. |

## Working locally

```bash
npm install
npm run dev
npm run build
```

## Usage guidance

Use the provided SVG assets whenever possible for crisp scaling. Always refer to the product as **Aspire**, not ".NET Aspire". Keep the logo proportions and fill colors intact, avoid introducing alternate lockups or unofficial variants, and point people to [aspire.dev](https://aspire.dev) for official docs and product content.
